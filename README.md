# The Last World BEP20 Contracts
Implement an upgradeable TLW bep20 contract and token vesting contract

## Install dependency

```shell script
npm install
```

## Run unittests

Start ganache:
```shell script
npm run testrpc
```

Run unittests:
```shell script
npm run truffle:test
```

## Flatten

```shell script
npm run flatten
```
